XTCEasyRootPlus 使用的文件

部分文件介绍:

```files
launchers.json: 桌面版本列表
superrecovery.json: 超级恢复文件列表
version.json: XTCEasyRootPlus 的版本文件
1userdata.img: Magisk25200 使用的 UserData 文件
1userdata.img: MagiskDelta25210 使用的 UserData 文件
设备型号.zip: 对应型号Root时所需要的文件
xposed-magisk-1.zip: Xposed 模块 1
xposed-magisk-2.zip: Xposed 模块 2
xtcpatch.zip: 小天才 8.1 机型使用的 XTCPatch 模块
xtcpatch2100.zip: 小天才 7.1.1 机型使用的 XTCPatch 模块
caremeospro.zip: CaremeOS Pro 模块
doze.zip: Doze 模块
```
